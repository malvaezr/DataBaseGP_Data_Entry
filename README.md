# J & R Steel Welding LLC - Website

Modern, responsive website for J & R Steel Welding LLC, a Houston-based metal decking installation company.

## Pages

- **Home** (`index.html`) - Company overview, features, and about section
- **Services** (`services.html`) - Detailed service breakdown (loading, laying, installing)
- **Contact** (`contact.html`) - Contact form and business information

## Tech Stack

- HTML5 with semantic markup
- CSS3 with custom properties and responsive design
- Vanilla JavaScript (no frameworks)
- Google Fonts (Inter)

## Running Locally

Open `index.html` in a browser, or use a local server:

```bash
# Python
python3 -m http.server 8000

# Node.js (if npx available)
npx serve .
```

Then visit `http://localhost:8000`.
